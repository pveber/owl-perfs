# Owl-perfs

This benchmark shows that while owl is as fast as lacaml for
operations on large matrices (dim = 100), it is slower by a factor 2
to 3 on small ones (dim = 4). Also vector initialization is slower
with owl whatever the size.

```
┌──────────────────────┬──────────┬─────────┬────────────┐
│ Name                 │ Time/Run │ mWd/Run │ Percentage │
├──────────────────────┼──────────┼─────────┼────────────┤
│ lacaml-mat-vec-mul-4 │ 194.78ns │  22.00w │     35.46% │
│ owl-mat-vec-mul-4    │ 549.35ns │ 148.00w │    100.00% │
└──────────────────────┴──────────┴─────────┴────────────┘
```
